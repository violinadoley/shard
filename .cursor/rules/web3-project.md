# Web3 Project Rules

## Smart Contracts

- Always verify contract addresses before interacting
- Never mock contract integrations with frontend
- Use appropriate testnets before mainnet deployment
- Never commit private keys or secrets

## Frontend

- Never mock backend/contract calls with fake data
- Always connect to real contract addresses
- Handle wallet disconnection gracefully
- Support multiple wallet providers (MetaMask, etc.)

## General

- Follow clean code principles
- Document all public functions
- Write tests for all critical paths
